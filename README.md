# jinmoncom2020-data

## 概要
[人文科学とコンピュータシンポジウム2020](http://jinmoncom.jp/sympo2020/) での発表

『日本語歴史コーパス』の文脈化単語埋め込みに基づく意味空間
浅原正幸・加藤祥

の関連データです。

[紹介動画](https://youtu.be/MfSLTztWEzw)

## 01_word_sense
分類語彙表番号が 23100 (用-活動-言語-言語活動)の単語ベクトルデータ

## 02_genji
源氏物語の帖別文ベクトルデータ

## 03_covar
分散・共分散行列のノルムを評価した語彙素に「赤い」を含む単語ベクトルデータ
